FOLLOWING ARE MOST ESSENTIAL COMMANDS FOR GIT & GITHUB.
1. GIT INIT
2. GIT STATUS
3. GIT ADD  [UNTRACKED FILE TO STAGED]
4. GIT COMMIT -M FILE NAME  [STAGED TO TRACK]
5. GIT RESTORE [RESTORE DELETE FILE]

PUSH CODE INTO GITHUB.
1. GIT REMOTE ADD ORIGIN https://github.com/user/repo.git
2. GIT PUSH ORIGIN MAIN [BRANCH NAME]

PULL UPDATED CODE FROM GITHUB TO LOCAL SYSTEM.
1. GIT PULL ORIGIN MAIN

CHECK LOGS AND CREATE NEW BRANCH.
1. GIT LOG
2. GIT LOG --ONELINE
3. GIT BRANCH
3. GIT SWITCH [BRANCH NAME]
